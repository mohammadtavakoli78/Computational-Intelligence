## Introduction
At This Project we ```Classify Six Fruits``` with the help of ```Nueral Networks```. I implemented ```Back Propagation``` and ```Feed Forward``` and cost functions like: ```SoftMax```, ```relu```, ```tanh``` without the help of libraries.<br>
